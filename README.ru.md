# Smart TV Media Player - Руководство по установке и запуску

[Read in English](README.md)

* **Требования**:
  * Установленный Python версии 3.8 или выше.
  * Операционная система Windows (для использования готовых скриптов автоматизации `.bat`).
  * Устройство Android TV, подключенное к той же сети Wi-Fi, что и ваш компьютер.
  * Включенная отладка по ADB и активированный режим разработчика (Developer Options) на телевизоре.

* **Настройка бэкенд-сервера**:
  * Запустите файл `start_server.bat` в корневой директории проекта.
  * Этот скрипт автоматически установит необходимые зависимости Python (Flask, Flask-SQLAlchemy, Flask-CORS, PyJWT, openpyxl, Pillow, static-ffmpeg) и запустит Flask-сервер.
  * Либо перейдите в директорию `server` и выполните команды вручную:
    * `pip install -r requirements.txt`
    * `python app.py`

* **Сборка приложения Android TV**:
  * Запустите файл `build_apk.bat` в корневой директории.
  * Этот скрипт настроит среду сборки, используя предустановленные JDK 17 и Android SDK.
  * Он скомпилирует код и сохранит готовый файл `SmartTVMediaPlayer.apk` в корневой директории.

* **Установка приложения Android TV**:
  * Запустите файл `install_apk_to_tv.bat` в корневой директории.
  * При запросе в консоли введите локальный IP-адрес вашего Android TV (например, `192.168.1.100`).
  * Подтвердите запрос на разрешение отладки на экране телевизора, когда он появится.
  * Скрипт установит APK на телевизор через ADB. После завершения установки вы найдете и запустите приложение "Smart TV Media Player" в списке установленных программ на вашем ТВ.

Приятного просмотра и управления медиафайлами на вашем Android TV!
